# Quiz-App-Using-CoreJava

 # Quiz Application (Java)

## 📌 Overview

This is a simple **Java Console Quiz Application** that asks users multiple-choice questions, accepts their answers, checks the responses, and displays the final score.

The project is designed using **Object-Oriented Programming (OOP)** concepts in Java.

---

## 🚀 Features

* Displays 6 multiple-choice questions.
* Accepts user input through the console.
* Validates answers.
* Shows whether each answer is correct or incorrect.
* Calculates and displays the final score.

---

## 🛠 Technologies Used

* Java
* OOP (Classes & Objects)
* Arrays
* Scanner Class

---

```
QuizApplication/

```

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/QuizApplication.git
```

2. Navigate to the project folder

```bash
cd QuizApplication
```

3. Compile the Java files

```bash
javac *.java
```

4. Run the application

```bash
java Main
```
